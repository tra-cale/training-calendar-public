# 🏃‍♂️ トレーニング管理カレンダー 2025

血圧・体重・トレーニング実績を総合的に管理する**無料のWebアプリケーション**です。
シンプルで使いやすく、あなたの健康管理をサポートします。

## 🌐 デモサイト

**🔗 今すぐ使ってみる**: [https://tra-cale.github.io/training-calendar-public/](https://tra-cale.github.io/training-calendar-public/)

インストール不要！ブラウザで開くだけで使えます。

## ✨ 主な機能

### 📊 健康データ管理
- **血圧記録**: 最高血圧・最低血圧の日次記録
- **体重管理**: 毎日の体重変化を追跡
- **グラフ表示**: 見やすいグラフで推移を確認

### 💪 トレーニング記録
- **5種類の運動**に対応
  - ウォーミングアップ
  - ランニング（速度別）
  - エアロバイク（負荷レベル別）
  - 筋トレ（5種目）
  - クールダウン
- **消費カロリー自動計算**（METs方式）
- **目標設定と達成度管理**

### 📱 どこでも使える
- **PC・タブレット・スマホ対応**
- **完全無料**
- **個人情報保護**: データは全てブラウザに保存

## 🚀 使い方

1. [デモサイト](https://tra-cale.github.io/training-calendar-public/)にアクセス
2. プロフィール情報を入力（性別、年齢、体重など）
3. カレンダーで日付をクリックしてデータを記録
4. 月間・年間サマリーで進捗を確認

詳しい使い方は[使い方ガイド](https://tra-cale.github.io/training-calendar-public/features.html)をご覧ください。

## 🔒 プライバシーとセキュリティ

- **データは全てあなたのブラウザに保存**されます
- **外部サーバーへの送信は一切ありません**
- **個人情報が漏洩する心配はありません**

⚠️ **注意**: ブラウザのデータを削除するとアプリのデータも消えます。
定期的にスクリーンショットを撮ることをお勧めします。

## 📱 スマホアプリのように使う

### iPhone/iPad
1. Safariで開く
2. 画面下の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」をタップ

### Android
1. Chromeで開く
2. メニュー（⋮）→「ホーム画面に追加」
